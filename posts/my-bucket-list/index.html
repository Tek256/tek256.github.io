my bucket list

Dec 4, 2019

One day I’m going to die. This is a universal truth for each and every one of us. The last few weeks before writing this have been rough for me. That’s why I’m writing this, as a message to my future self to recognize where I was and where I’ve gotten to. I encourage everyone to write a bucket list of their own.

Here are the current goals of mine, I won’t remove any, but I will add some if I start to run low.

  • Build my own house
  • Make a difference
  • Fall in love
  • Make a game completely on my own
  • Be successful working for myself
  • Make a documentary
  • Create a meaningful podcast (series, limited run)
  • Make peace with my depression
  • Build an accessible open source game console
  • Paint a mural
  • Travel to Scotland
  • Write a book
  • Make a zine
  • Go Sky Diving
  • Put on an EDM Show
  • Master Bartending
  • Read 100 books
  • Roadtrip across the US
  • Take up boxing
  • Help someone achieve a dream
  • Mentor someone
  • Learn about someone’s life
  • Make a game that had an impact
  • Share my life story with others
  • General
  • Personal
